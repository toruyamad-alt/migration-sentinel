# Pricing

## Suggested launch pricing

- Free
  Public repositories, up to 100 migration scans per month, default rules only
- Team
  `$39` per repository per month, private repositories, GitHub annotations, Slack webhook, custom config
- Scale
  `$199` per organization per month, unlimited repositories, audit history, policy bundles, priority support

## Why teams would pay

- A single migration outage can cost far more than a monthly tool subscription.
- The product blocks a narrow but painful class of production incidents.
- Teams can trial it on one repository before wider rollout.

## Expansion path

- Add paid rule packs by framework: Rails, Prisma, Laravel, TypeORM
- Add hosted policy sync and incident history
- Add organization-level exemptions and approval workflow
