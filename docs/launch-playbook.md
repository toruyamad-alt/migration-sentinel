# Launch Playbook

## What is already done

- CLI scanner implemented
- GitHub Action metadata added
- CI workflow added
- Release workflow added
- Pricing and listing copy drafted

## Steps that still require your accounts

1. Create a GitHub repository and push this project.
2. Replace placeholder repository URLs in `package.json`.
3. Add a tag like `v0.1.0` to trigger a release.
4. Publish the action in GitHub Marketplace.
5. Add billing if you want private-repo paid plans.

## Fastest path to first dollar

1. Launch with a free public-repo tier.
2. Offer private-repo access manually to 3-5 early teams.
3. Add one premium framework pack and one hosted feature.
4. Convert those manual customers into a flat monthly plan.
